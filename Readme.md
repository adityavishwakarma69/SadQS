# SadQS

A Quickshell desktop shell, which makes me sad :( because writing it is so tiring

---

## Current Features
#### These are the list of features
### widgets
- Bar.
- Common Modules (Widget, StaticWidget, ExpandableWidget, Slider).
### services
- NiriSvc (uses [qml-niri](https://github.com/imiric/qml-niri)).
- NetworkSvc (Uses [Quickshell.Networking](https://quickshell.org/docs/v0.3.0/types/Quickshell.Networking)) this adds a live bind to primary network device which the base service doesn't provide (works only with NetworkManager).
- BrightnessSvc (uses [Quickshell.Io](https://quickshell.org/docs/v0.3.0/types/Quickshell.Io)) only works with Brightnessctl
